# Kakudite

A kakoune config for the curious. 

# Installation

To install, run:

```bash
git clone https://github.com/uncomfyhalomacro/kakudite ~/.config/kak
```

Then finally run

```bash
kak -e 'bundle-install'
```

# Requires

- xplr
- notify-send
- rust
- zellij
- julia or juliaup
- GNU parallel

## Bootstrapped with kak-bundle

- kak-tree-sitter
- kak-lsp
